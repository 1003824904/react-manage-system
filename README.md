# React Manage System
基于React + Ant Design 的后台管理系统解决方案。( Vue + Element 版本请看 [vue-manage-system](https://github.com/lin-xin/vue-manage-system) )

## 功能
- [x] Ant Design of React
- [x] 登录/注销
- [x] Dashboard
- [x] 表格
- [x] Tab选项卡
- [x] 表单
- [x] 图表 :bar_chart:
- [ ] 富文本编辑器
- [ ] markdown编辑器
- [x] 图片拖拽/裁剪上传
- [ ] 支持切换主题色 :sparkles:
- [ ] 列表拖拽排序
- [ ] 权限测试
- [ ] 404 / 403
- [X] 三级菜单
- [x] 自定义图标
- [ ] 可拖拽弹窗
- [ ] 国际化
